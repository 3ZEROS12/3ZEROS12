## Featured Project

### Digit Recognition FastAPI

A Python project built around digit recognition, image preprocessing, and API deployment. It includes an MNIST CNN training script, a FastAPI inference service, and OpenCV preprocessing for single-digit, multi-digit, and seal-region recognition scenarios.

What it demonstrates:

- Training and loading a PyTorch CNN model
- Using OpenCV for image preprocessing, red-region filtering, skew correction, and character segmentation
- Building an upload-based inference endpoint with FastAPI
- Documenting a technical prototype so other users can understand and run it

Repository: https://github.com/3ZEROS12/digit-recognition-fastapi
